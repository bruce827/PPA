# 福州码头智慧管控平台建设项目软件功能概述
本项目聚焦福州码头煤炭存储与装卸、铁矿石运输核心业务，围绕“安全、高效、绿色、智慧”管控目标，构建覆盖数据中枢、业务管理、设备运维、智能监测的一体化软件平台，包含**4个核心业务系统**与**1个统一系统底座**，各模块协同实现码头业务“自动化、集成化、智能化、平台化”转型，具体功能概述如下：


## 一、系统平台底座（统一支撑中枢）
作为全平台技术支撑核心，提供跨系统协同、统一管控能力，避免“信息孤岛”，核心功能包括：
1. **统一工作台**：打造“一站式”操作门户，集成生产KPI（吞吐量、设备故障率）、人力资源状态（关键岗位在岗情况）、环境数据（气象、温湿度）的个性化仪表盘；归集各系统待办任务（设备维修审批、生产计划确认），支持按紧急程度分类、一键跳转处理，超期任务自动催办；兼容PC端与移动端（微信小程序/H5），界面自适应不同终端。
2. **统一身份与组织管理**：对接集团4A统一身份认证系统，实现账号密码、短信验证码双因子登录；维护码头内部部门（生产部、设备部）与外部供应商层级结构，同步4A用户信息（每日全量+实时增量），人员岗位变更时自动更新平台权限，杜绝越权访问。
3. **统一流程与待办管理**：可视化配置业务流程（如数据资产申请、API调用审批），支持并行/排他网关等复杂流程；集中接收各系统待办数据（含来源、紧急程度、处理链接），异常格式数据自动拒收告警，历史待办查询周期≥3年。


## 二、核心业务系统功能
### （一）数据融合平台（全量数据中枢）
作为数据“汇聚-治理-共享-应用”核心载体，打通多系统数据链路，支撑业务决策，核心功能包括：
1. **多源数据集成**：兼容工业协议（OPC UA、Modbus TCP）接入生产设备实时数据（卸船机转速、皮带电流）、ERP接口（SAP OData）接入集团经营数据（设备台账、物资库存）、API/FTPS接入天津港务安全环保数据（粉尘浓度、作业票），实现协议转换与数据标准化清洗。
2. **数据资产与共享**：按“生产/设备/安全/经营/项目”划分数据目录，对生产计划、设备台账等核心数据进行资产化登记，支持检索、申请使用；通过HTTPS加密API对外提供数据共享服务（如向生产系统推送设备状态），支持OAuth 2.0认证、调用量监控与日志审计。
3. **可视化分析**：构建全局业务视图（跨域指标联动，如设备告警关联生产中断记录）、生产运行监控（船舶作业状态、库场库存趋势）、设备效能分析（故障率、维护成本趋势），支持多维度下钻与图表导出（PDF/Excel），核心指标实时更新延迟≤5秒。


### （二）生产管理系统（业务全流程管控）
覆盖码头生产“商务-计划-调度-库场”全链路，规范业务流程、提升作业效率，核心功能包括：
1. **商务与计划管理**：管理客户档案与合同关联关系（年度/单船/临时客户），动态配置费率体系（合同/标准费率），打通计费-应收-开票-收款全流程；构建“年→月→日”三级生产计划体系，覆盖船舶卸船/装船预报、动态计划、执行计划，关键节点自动提醒，支持计划修订与班组任务派发。
2. **多式联运调度**：实现船舶作业全流程管控（待泊-靠泊-卸船/装船-离港），同步船舶动态数据；协调铁路/汽运陆运计划（火车到港统计、汽运作业确认）、电厂输煤皮带调度（作业联动与执行反馈）；管理港口配套服务（岸电供应、采样服务），形成调度凭证闭环。
3. **库场与统计分析**：可视化监控库场堆存空间（垛位占用、货物流转），自动生成船舶关联库存报表；构建多维度统计体系，包括作业效率（锚地泊位时效）、仓储动态（库场周转率）、生产中断原因归集，支持日报/周报/年报自动生成。


### （三）设备全寿命周期管理系统（运维智能化升级）
覆盖设备“台账-运维-成本-报废”全生命周期，联动集团ERP实现精益化管理，核心功能包括：
1. **设备基础与运维管理**：建立设备台账（含BOM清单、技术图纸），对接ERP PM模块同步设备主数据；制定点巡检计划（按路线/点位配置检查标准），支持手持终端（NFC）执行巡检、异常实时触发缺陷流程；管理检修计划（年/月/周）、保养维护（标准作业指导书）、检测检验（特种设备资质关联），维修工单状态同步至ERP。
2. **故障与成本分析**：实现缺陷/故障“上报-处置-整改”闭环，自动关联历史维修记录；统计设备能耗（油/电/水）、维护成本（备件、人工），生成设备能效分析报表（关键指标如设备可用率=（运行时间-故障时间）/运行时间）；管理大型部件（可修复部件外委维修、报废判定）与备品备件（库存预警、需求计划）。
3. **应急与合规管理**：维护设备应急预案（如皮带跑偏处置流程）、安全许可证（特种设备年检）；记录设备运行日报、劳务工管理信息，生成设备运行关键绩效指标（KPI）报表，支撑运维决策优化。


### （四）智能视频分析系统（安全与设备智能监测）
基于150路摄像头、300路算法计算，实现“人员行为合规+设备状态异常”实时识别，核心功能包括：
1. **人员行为监控**：覆盖码头关键区域（变电所、转运站），识别人员安全帽佩戴、工装穿戴合规性，未达标时2秒内触发报警；监测人员定时巡检（未按规定时间到岗）、抽烟、跌倒、脱岗、违规穿越电子边界等风险行为，报警信息同步推送至数据融合平台。
2. **设备状态识别**：聚焦皮带机、除铁器等核心设备，识别皮带跑偏（偏移量≥15%时准确率≥70%）、除铁器异物（≥30cm异物准确率≥70%）、皮带堵煤（堆积高度≥60cm准确率≥70%）、皮带异物（≥45cm块状异物准确率≥70%）；适配高粉尘、强光照环境，通过图像预处理（去噪、增强对比度）优化识别精度，报警响应延迟≤2秒。
3. **联动与统计**：支持自定义报警联动规则（如识别皮带跑偏后提醒操作员）；按算法类型、区域、时间维度统计报警数据，生成月度安全风险报表，历史识别记录留存≥3年。


## 三、整体功能价值
1. **数据打通**：整合16类系统数据（生产/设备/ERP/安全环保），数据传输时效达标（生产数据延迟≤1秒，跨地域数据错误重传≥3次），为决策提供全量数据支撑；
2. **效率提升**：生产计划自动分解、设备巡检移动端执行、待办任务集中处理，减少人工操作环节，预计码头作业效率提升15%-20%；
3. **风险预警**：智能视频分析提前识别人员违规、设备异常，安全环保数据实时监控，降低事故发生率；
4. **合规管控**：遵循集团信息化标准与国家等保三级要求，数据传输加密（SSL/TLS 1.2+）、操作日志留存≥6个月，满足网络安全、数据安全合规要求。
