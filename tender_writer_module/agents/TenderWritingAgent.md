# Agent: TenderWritingAgent

## Description

This agent is the master coordinator for the `TenderWriter` module. It orchestrates the entire process of generating a tender document, from analyzing initial requirements to producing the final, structured output.

## Responsibilities

-   Executes the `GenerateTenderDocument` workflow.
-   Manages the `GenerateOutlineTask` and `GenerateContentTask`.
-   Handles user interactions for generating the table of contents and image assets.
-   Coordinates the flow of data between different tasks and components.
