# 投资人如何找项目

## 1. 文档目的

这份文档不是“创业者去哪里曝光”的泛平台清单，而是从投资人视角出发，整理更像真实 `deal flow` 的项目发现渠道。

核心问题是：

**如果我是一个早期投资人，我通常会去哪些平台、社区和场域寻找更靠谱的创业项目？**

## 2. 核心结论

靠谱投资人通常不会把主要精力放在泛内容平台上找项目。

他们更常使用的是一套分层 `deal flow`：

1. 加速器 / 创业营 / Demo Day
2. 高质量创业者社区
3. 创投撮合与项目平台
4. 创投数据库与研究工具
5. 活动网络
6. 内容平台

所以从创业者视角看，想被投资人更快看到，优先级应当是：

**先进入投资人已经在看的入口，再考虑内容平台放大。**

## 3. 一级渠道：加速器 / 创业营 / Demo Day

这是最重要的一层，因为它天然带有预筛选和信誉背书。

### 典型平台 / 场域

- `Y Combinator Demo Day`
- `Techstars Demo Day`
- `MiraclePlus / 奇绩创坛`

### 为什么重要

- 投资人默认这些项目已经经过一次筛选
- 单位时间内高密度出现优秀项目
- 更容易建立 warm intro
- 更容易被快速判断是否值得继续看

### 对创业者的启发

- 如果项目适合，尽早进入这类体系
- 即使没有马上被投，也能获得高密度反馈

## 4. 二级渠道：高质量创业者社区

这层不是传统融资平台，但往往最早出现值得看的 founder 和项目。

### 典型平台 / 场域

- `Founder Park`
- `Indie Hackers`
- `Product Hunt`
- `脉脉`

### 为什么重要

- 投资人可以先看 founder 质量
- 可以观察项目叙事、执行力和迭代速度
- 社区关系链容易形成 warm intro

### 对创业者的启发

- 先进入高质量创业者社区，而不是只做内容曝光
- 让别人能看到你的产品认知、迭代记录和行业洞察

## 5. 三级渠道：创投撮合与项目平台

这层是更直接的项目对接场。

### 典型平台 / 场域

- `创业邦 DEMO CHINA`
- `36氪创投平台`
- `投融界`
- `企名片`

### 为什么重要

- 创业项目和投资机构集中出现
- 适合初步建立连接
- 更适合中国语境下的早期对接

### 对创业者的启发

- 如果你还没有强人脉，这是现实可用的起点
- 适合作为补充渠道，而不是唯一渠道

## 6. 四级渠道：创投数据库与研究工具

这层不是社交入口，而是系统化发现项目的工具层。

### 典型平台 / 场域

- `Crunchbase`
- `AngelList / Meridian`

### 为什么重要

- 方便投资人筛选赛道、地域、阶段和融资动态
- 适合做广覆盖搜索和二级研究

### 对创业者的启发

- 当项目开始对外融资或全球化曝光时，这层会越来越重要

## 7. 五级渠道：活动网络

活动平台本身不是创投机构，但它是人与人高密度连接的发生器。

### 典型平台 / 场域

- `活动行`

### 为什么重要

- 创业活动、AI 活动、路演、闭门会、产业对接都密集发生
- 对没有既有人脉的创业者尤其有价值

### 对创业者的启发

- 不是为了“报名活动”本身
- 是为了进入更多能形成后续跟进的场域

## 8. 六级渠道：内容平台

内容平台是认知放大器，但通常不是主 `deal flow`。

### 典型平台 / 场域

- `小红书`
- `Bilibili`
- `小宇宙`

### 为什么不是主渠道

- 关系密度低
- 投资决策链条长
- 更适合建立 founder 形象和认知势能，而不是快速建立投资连接

### 对创业者的启发

- 可以做，但优先级不要高于前面几层
- 适合做 founder log、产品认知、市场观察和长期品牌

## 9. 从投资人视角的优先级排序

如果站在投资人视角，我会优先看：

1. 有预筛选的创业营和 Demo Day
2. 高质量 founder 社区
3. 创投撮合平台和项目平台
4. 数据库和研究工具
5. 活动网络
6. 内容平台

## 10. 从创业者视角的行动含义

如果你想更快被投资人看到，不应只问：

**“我要不要做内容？”**

而应该先问：

**“我要不要进入投资人已经在看的渠道？”**

对 `PPA 2.0` 当前阶段，更现实的顺序是：

1. 进入创业营 / Demo Day / 路演体系
2. 进入高质量创业者社区
3. 建立创投平台和活动网络上的存在感
4. 用内容平台做认知放大，而不是把内容平台当唯一渠道

## 11. 对 PPA 2.0 的具体建议

当前更值得关注的，不是“广撒网到所有平台”，而是：

- 有没有可能进入 `MiraclePlus / 奇绩创坛` 这类强筛选体系
- 能不能进入 `Founder Park` 这类高质量创业者社区
- 能不能在 `创业邦 DEMO CHINA`、`36氪创投平台`、`投融界` 里建立项目存在
- 能不能通过 `活动行` 找到 AI / 创业 / 路演活动形成线下连接
- 能不能通过 `脉脉` 找到相关 founder、投资经理、FA、行业资源方

## 12. 一句话结论

**内容平台是放大器，创业者社区和投资人 already watch 的 deal flow 才是连接器。**
