## 四、 落地核心代码与配置模板

### 1. 资产脱敏与 Frontmatter 标签自动化处理脚本（Python）
```python
import os
import subprocess

def process_and_tag_document(docx_path, output_dir, proj_name, download_url):
    """
    1. 使用 pandoc 转换 word 为带有标准表格的 markdown
    2. 在 markdown 头部写入 frontmatter 数据，供 VitePress 渲染和 RAG 检索
    """
    file_name = os.path.basename(docx_path)
    base_name = os.path.splitext(file_name)[0]
    output_md_path = os.path.join(output_dir, f"{base_name}.md")
    
    # 调用 Pandoc 进行转换
    subprocess.run([
        "pandoc", docx_path, 
        "-f", "docx", 
        "-t", "markdown_github", 
        "-o", output_md_path
    ], check=True)
    
    # 读取内容并注入 Frontmatter 标签
    with open(output_md_path, 'r', encoding='utf-8') as f:
        content = f.read()
        
    frontmatter = f"""---
title: {base_name}
project_name: {proj_name}
download_link: {download_url}
---

::: info 物理文件下载提示
本节内容对应的完整原始打包工程文件，您可以 **[👉 点击此处安全下载原件]({download_url})** 进行二次修改与参考。
:::

"""
    with open(output_md_path, 'w', encoding='utf-8') as f:
        f.write(frontmatter + content)
        
    print(f"成功处理并添加元数据：{output_md_path}")
```
