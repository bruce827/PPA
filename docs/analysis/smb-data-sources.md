# 小微企业与目标客群数据来源清单

本文件汇总与你“小微企业软件项目评估 / PPA 评估引擎”相关的高价值数据来源，方便后续自行调研与报告写作。可与 `docs/analysis/smb-archetypes.md` 配合使用。

> 说明：本清单只给出机构与入口方向，不含具体统计数字与网址，避免信息过期。使用时建议优先查找最新年度（或最近一期）正式报告与公报。

---

## 1. 小微企业整体规模 / 倒闭 / 行业分布 / 金融风险

### 1.1 定义与总体规模

- **工业和信息化部（工信部）中小企业局**  
  - 关注文档：  
    - 《中小企业划型标准规定》（最新版）  
    - 各年度中小企业发展报告 / 情况简报  
  - 用途：  
    - 明确“小微企业”在不同产业的官方划型口径  
    - 为报告中的“小微企业定义”提供权威依据

- **国家统计局**  
  - 关注文档：  
    - 全国经济普查公报  
    - 年度统计公报中与“第三产业 / 信息传输、软件和信息技术服务业 / 教育 / 租赁和商务服务业”等相关部分  
  - 用途：  
    - 获取按行业划分的企业数量、从业人员、增加值等宏观结构数据  
    - 支撑“整体盘子有多大”的判断

### 1.2 存量 / 注销 / 倒闭数量

- **国家市场监督管理总局（企业登记）**  
  - 关注内容：  
    - 全国产业主体登记统计（新设、在册、注销等）  
    - 各省市市场监管局发布的企业登记年报  
  - 用途：  
    - 获取“小微企业注册量 / 注销量”的时间序列数据  
    - 结合行业划分，分析哪些行业退出率较高  
  - 深度获取方式：  
    - 通过“政府信息公开申请”获取更细粒度数据（如按行业、规模分组的注销数）

- **国家企业信用信息公示系统**  
  - 关注内容：  
    - 在册企业、吊销、注销的公示信息  
  - 用途：  
    - 针对某城市 / 某行业做样本筛选，观察存续状态与退出情况

### 1.3 破产数据

- **最高人民法院 / 各省高级人民法院破产审判庭**  
  - 关注文档：  
    - 《全国法院破产审判工作白皮书》  
    - 各省高院发布的破产审判年度报告  
  - 用途：  
    - 把“注销”中的正常退出与司法破产区分开来  
    - 了解破产案件的行业分布、成因与趋势

### 1.4 金融风险与小微信贷

- **中国银保监会及各地银保监局**  
  - 关注文档：  
    - 小微企业金融服务相关专项报告  
    - 地方“小微企业金融服务情况”新闻发布材料  
  - 用途：  
    - 了解小微企业融资难易度、信贷投放、风险暴露情况  
    - 支撑对“小微企业现金流压力与抗风险能力”的判断

- **中国人民银行 / 地方人民银行分支机构**  
  - 关注内容：  
    - 金融统计数据与普惠金融相关报告  
  - 用途：  
    - 补充小微企业融资成本、授信结构等信息

## 2. 三类目标小微企业的行业与数量数据

与你在 `smb-archetypes.md` 中定义的三类目标客群对应：

1. 困境中的小型软件公司  
2. 本地教育培训机构  
3. 专业服务小机构（会计所 / 律所 / 咨询等）

### 2.1 小型软件公司

- **工信部电子信息司 / 软件服务业司**  
  - 关注文档：  
    - 《软件和信息技术服务业发展报告 / 统计公报》  
  - 用途：  
    - 获取软件企业数量、收入规模、企业规模结构等数据  
    - 观察中小型软件企业的数量与发展趋势

- **中国软件行业协会等行业组织**  
  - 关注内容：  
    - 行业年报、白皮书、景气度调查  
  - 用途：  
    - 补充民间视角的数据与对景气度、竞争格局的分析

### 2.2 本地教育培训机构

- **教育部 / 各省市教育厅（局）**  
  - 关注文档：  
    - 民办教育机构登记情况通报  
    - 教培行业整治和整改总结报告  
  - 用途：  
    - 了解民办教育机构的数量、类型（学科类/非学科类）、地区分布  
    - 把握政策对教培机构存量与增量的影响

- **教育行业协会 / 民办教育协会**  
  - 关注内容：  
    - 民办教育 / 教培领域蓝皮书、年度报告  
  - 用途：  
    - 获取更贴近一线机构生存状态的行业观察

### 2.3 专业服务小机构（会计 / 律所 / 咨询等）

- **会计师事务所**  
  - 主管与行业组织：财政部、中国注册会计师协会  
  - 关注文档：  
    - 会计师事务所行业统计数据、发展报告  
  - 用途：  
    - 获取事务所数量、规模结构、行业集中度等信息

- **律师事务所**  
  - 主管部门：司法部 / 各地司法厅局  
  - 关注内容：  
    - 律师事务所统计公报、工作年鉴  
  - 用途：  
    - 了解律所数量、规模与区域分布，为“专业服务类小机构”做基准

- **咨询、猎头、人力资源服务机构**  
  - 主管部门与行业协会：人社部门、商务部门及相关行业协会  
  - 关注内容：  
    - 行业年鉴、发展报告、白皮书  
  - 用途：  
    - 把握专业服务类小机构的整体盘子与典型业务模式

## 3. 与软件项目直接相关的需求与项目数据

### 3.1 招投标项目（小软件公司 archetype）

- **中国招标投标公共服务平台**  
  - 关注内容：  
    - 全国软件开发、信息系统集成等项目的招标公告与中标公示  
  - 用途：  
    - 分析 IT 项目在金额、周期、技术要求上的分布  
    - 观察小型软件公司常见的项目类型与价格区间

- **各省 / 地市公共资源交易中心**  
  - 用途：  
    - 聚焦特定区域（如辽宁 / 沈阳）的政府与国企信息化项目  
    - 形成“本地软件公司可接触标的”的结构样本

### 3.2 现有 SaaS / 软件方案（教培与专业服务 archetype）

- **教培 SaaS 供应商官网**  
  - 关注内容：  
    - 排课、收费、家校沟通一体化平台的功能列表与价格方案  
  - 用途：  
    - 清楚教培 SaaS 能力上限与典型定价，为“定制 vs SaaS”评估提供对照

- **律所 / 财税 SaaS、通用 CRM / 项目管理工具**  
  - 关注内容：  
    - 针对专业服务行业的垂直产品  
    - 通用型项目管理与客户管理工具的行业解决方案案例  
  - 用途：  
    - 判断“小机构是否真的需要定制系统”，还是优先选型与配置即可

## 4. 企业层面的结构性样本（辅助分析）

- **企查查 / 天眼查 / 爱企查等商业企业信息平台**  
  - 关注内容：  
    - 按城市 + 行业 + 规模筛选的企业列表  
    - 存续状态、成立时间、注册资本区间等基础信息  
  - 用途：  
    - 为三类目标客群构建“结构性样本”：  
      - 如“沈阳 + 软件开发 + 小微”  
      - “沈阳 + 教育咨询/培训”  
      - “沈阳 + 法律服务 / 财务咨询”  
    - 帮助验证官方统计的结构与一线情况是否一致

## 5. 建议优先查阅的高价值文档类型

在调研起步阶段，可优先查找以下类型文档（以最新年份为主）：

1. 《中小企业划型标准规定》（工信部）  
2. 最新一份全国经济普查公报（特别是服务业相关章节）  
3. 最近 3–5 年的《全国法院破产审判工作白皮书》（最高法）  
4. 最近 1–2 份银保监会关于“小微企业金融服务”的专项报告（全国或东北/辽宁地区）  
5. 《软件和信息技术服务业统计公报 / 发展报告》（工信部或中国软件行业协会）  
6. 教育部 / 地方教育部门发布的民办教育机构统计与政策执行总结  
7. 中国注册会计师协会、司法部等披露的会计师事务所与律师事务所统计资料

这些文档可以作为你撰写《小微企业软件项目需求与评估机会》研究报告的宏观数据基础，再与 `smb-archetypes.md` 中的三个具体场景结合，形成“宏观 + 典型场景”的完整故事线。
