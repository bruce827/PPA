# 项目机会智能评估服务文档包

## 文档说明

本目录用于沉淀“项目招标文件自动筛选与推荐”第一阶段方案的统一过程文档。

该方案不是单一模型开发，而是一个可被 PPA 内部复用、也可对外提供接口的“项目机会智能评估服务”。

## 当前统一结论

1. 输入端由现有爬虫系统负责，全网招标信息采集不纳入本阶段实现范围。
2. 新能力核心是一个独立服务，不是单纯的 UI 页面，也不是单纯的 prompt 工作流。
3. 外部 AI 主要负责文档理解、图文识别、标签归一、解释生成，不直接承担最终推荐裁决。
4. 最终推荐结果必须由“结构化数据 + 检索结果 + 规则评分”共同产生。
5. 向量是辅助检索能力，不是最终业务真相；最终权威结果必须是结构化输出。
6. 服务设计需同时支持：
   - 被当前 PPA 后端调用
   - 未来独立系统调用
   - 对外 API 调用
