# PPA 3.0 文档目录

本目录用于存放 `PPA 3.0` 的独立产品规划与需求文档。

## 使用原则

- 与 `docs/prd/`、`docs/prd2.0/` 完全分离
- 本目录只承载 `3.0` 的长期平台化、网络效应和成熟商业模式相关设计

## 建议放置内容

- `vision.md`
- `product-brief.md`
- `prd.md`
- `architecture.md`
- `business-model.md`
- 其他与 `3.0` 直接相关的专题文档

## 当前定位

`PPA 3.0` 建议作为“成熟平台版”的版本空间，重点讨论：

- 更完整的社区与激励体系
- 专家评审与反作弊机制
- 企业化订阅或交易能力
- 更稳定的网络效应与规模化运营
