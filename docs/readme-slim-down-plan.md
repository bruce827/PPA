### 阶段 2：CLAUDE.md 重构（优先级：🔴 高）

**当前问题**：
- ❌ CLAUDE.md 太像 README（包含项目概述、技术栈、开发命令）
- ❌ 信息密度低，AI 需要扫描很多不相关的文字

**改造目标**：
- ✅ 只保留 AI 开发时**必须知道**的约束和约定
- ✅ 删除项目介绍（移到 README）
- ✅ 删除详细命令（改为"关键路径"）

**具体操作**：
1. 保留顶部"用中文回答"等个人指令
2. 删除"项目概述"段落
3. 删除"技术栈"表格（移到 README）
4. 删除"开发命令"的详细步骤
5. 改为"关键路径"速查表：
   ```markdown
   ## 📍 关键路径
   - 后端入口：`server/index.js`
   - 前端入口：`frontend/ppa_frontend/.umirc.ts`
   - 数据库：`server/ppa.db` (SQLite)
   - 计算引擎：`server/services/calculation.js`
   - AI 服务：`server/services/aiService.js`
   ```
6. 新增"🚨 强制约定"区块：
   ```markdown
   ## 🚨 强制约定
   - 前端必须用 **yarn**，禁止用 npm
   - 后端用 npm
   - SQL 字符串拼接（未参数化），存在注入风险 ⚠️
   ```
7. 新增"🏗️ 架构约束"区块：
   ```markdown
   ## 🏗️ 架构约束
   - 后端三层架构：Routes → Controllers → Services → Models
   - 新增功能必须走三层，禁止在 Routes 层写业务逻辑
   ```
8. 保留"已知限制"（改为"⚠️ 已知技术债"）
