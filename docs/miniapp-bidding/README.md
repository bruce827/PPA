## 当前结论

- 产品形态是微信小程序，不是现有 Umi 前端里的一个 Web 页面
- 数据源来自 PPA，但小程序首版不直接依赖本地 `server/`
- 首版后端采用微信云开发 CloudBase，以最快完成业务验证
- 所有微信登录用户都可查看招标信息
- 业务核心只有三件事：查看列表、查看详情、执行采纳

## 开发边界

- 本文档覆盖 MVP 首版
- 招标数据推送程序不在本次实施范围内，但接口边界已预留
- 小程序富文本公告只保证结构化展示，不承诺浏览器级样式还原
- 为了演示流畅性，PPA 侧将新增本地 staging 页面，不直接从前端读取 JSON 文件
