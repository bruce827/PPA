### 设计维度与逻辑说明：

1.  **核心成本维度拆解：**
    *   **AI服务费用**：作为项目的核心能力，占据了成本的大头（生产环境约10,000元/月）[1]。
    *   **基础设施费用**：包含了参考资料中提到的腾讯云服务器（7497.6元/年）、RDS数据库（7269.12元/年）以及域名解析等基础支出 [4]。
    *   **提效工具成本**：特别列出了老板关注的“先进生产力”投入，如 Cursor Ultra 会员（8400元）和开发期 API 费用 [2][4]。
    *   **运维与隐性成本**：将年度 30 人天的运维投入货币化体现，确保老板意识到后期维护的价值 [1]。

2.  **老板视角的“高阶数据”设计：**
    *   **数值卡片（KPI）**：直接区分“现金投入”与“人天投入”。资料中强调了“利用闲置人力不产生额外人力成本”，这对老板来说是极大的资源优化亮点 [1]。
    *   **ROI 预测标签**：直接给出结论——“命中3-5个项目即可回本”。这种**业务语言**比单纯的财务数字更能驱动决策 [1][3]。
    *   **风险与策略关联**：虽然页面主要展示成本，但通过“按量计费”等标签暗示了成本的可控性（如爬虫监控和AI按量付费） [1][5]。

3.  **移动端适配优化：**
    *   使用**环形图**而非传统的饼图，中心留白可在交互时展示详细金额，适合小屏操作。
    *   采用**响应式卡片布局**，确保在手机端浏览时无需横向滚动，关键财务信息（17.0万元）处于首屏视觉中心。
